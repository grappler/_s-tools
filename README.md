## Installation

Install grunt `npm install`
Run the grunt task `grunt css` and `grunt js` to compile and minify the scripts and styles.

Add this to the `functions.php`
```php
/**
 * Include Bootstrap
 */
require get_template_directory() . '/bootstrap.php';
```
